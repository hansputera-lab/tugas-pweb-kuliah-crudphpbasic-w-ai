# HRIS Installers

Cross-platform installers for deploying HRIS without manual setup.

## Windows

A GUI installer built with **Inno Setup** that bundles portable Apache, PHP, MariaDB, and the pre-built HRIS app. Users click through the wizard and get a fully working installation with Start Menu shortcuts.

See [windows/README.md](windows/README.md) for build instructions.

## Linux

A bash installer that supports Ubuntu, Debian, CentOS, RHEL, Fedora, Arch, and openSUSE. Runs interactively or headless via environment variables.

See [linux/README.md](linux/README.md) for usage.
